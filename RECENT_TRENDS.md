# 정보처리기사 실기 최근 5년 출제 흐름 반영 기준

## 분석 범위와 주의점

- 분석 범위는 2021~2025년의 개편 이후 시험 흐름이다.
- 시험 범위와 시행 정보의 기준점은 [Q-Net 정보처리기사 종목 정보](https://www.q-net.or.kr/crf005.do?gbnn=gbnSubtab2&id=crf00503&jmCd=1320)로 삼았다.
- 실기 문제는 시중의 복원·분석 자료를 교차 참고했다. 복원 문제는 실제 시험지와 세부 표현이 다를 수 있으므로 이 프로젝트는 기출 원문을 복제하지 않고 같은 역량을 묻는 새 문제를 사용한다.

## 확인한 흐름

1. 프로그래밍 언어 활용이 가장 큰 비중을 차지한다.
   - 2020년부터 2024년 1회까지의 복원 통계에서는 프로그래밍 언어 활용이 27%로 가장 높고, 응용 SW 기초 기술 16%, 보안 12%, SQL과 테스트가 각각 10%로 분석됐다.
   - 2023~2024년 회차별 정리에서도 C, Java, Python 문제가 반복적으로 높은 비중을 차지한다.
2. 코드 문제는 문법 암기보다 실행 결과 추적 능력을 자주 요구한다.
   - C의 포인터·구조체·비트·증감 연산, Java의 상속·다형성·예외, Python의 컬렉션·슬라이싱·상속을 우선 범위로 잡았다.
3. SQL과 데이터베이스는 매 회차를 대비해야 하는 안정적인 득점 영역이다.
   - SELECT, JOIN, GROUP BY/HAVING, NULL, DDL·DCL과 함께 정규화, 키, 무결성, 트랜잭션을 포함했다.
4. 코드만으로는 충분하지 않다.
   - 2025년 3회 복원 분석에서는 프로그래밍형과 용어형이 각각 8문제로 정리될 만큼 보안, 네트워크, 운영체제, 테스트, UML 등의 단답형도 중요했다.
5. 짧은 보기와 비교 학습을 함께 사용한다.
   - 실제 필답형 적응을 위해 정답은 직접 떠올리게 하되, 복습 모드에서는 비슷한 개념을 보기로 비교하도록 구성했다.
